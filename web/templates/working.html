Still working on it ...

Byzantium Linux is unable to automatically configure itself so far.

The autoconfiguration script is still trying and this page will auto-refresh every 60 seconds until it finishes.

In the event this page doesn't go away after many minutes:

Chances are, your computer has a wireless chipset that we don't have drivers or firmware for yet, or has drivers or firmware that don't support one or more functions we require. This is a problem.

There are lots of things you could do to troubleshoot, but if this is an emergency chances are you don't have a lot of time. If you have a USB wi-fi dongle, plug it into this computer and reboot Byzantium Linux. We support lots of USB wireless devices so chances are our driver kit will detect and configure it. When you have time (and internet) later please file a trouble ticket and we'll try to fix it in the next release.

Regardless, you have a fully functional desktop at your fingertips, so you can plug USB drives in, copy and edit files, and do other workstation things (just click the icon in the bottom-left corner). You can also plug mobile devices into this computer to recharge them.
